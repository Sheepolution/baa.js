Class Detail
baa.tween(obj, rate, vars, manager)
Making transitions of the value of properties easier. Make objects move from one point to another, use delay, easing, and call multiple tweens in a row. It is highly recommended that you use a tweenmanager.
Parameters:
{object} obj
The object you want to tween.
{number} rate
How long the tween should take in seconds.
{object} vars
An object with properties of the object, and the values you want them to tween to.
{object} manager Optional
The manager that owns this tween. Should only be given by the manager itself.
See:
baa.tweenManager
